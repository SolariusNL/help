# Support

Looking for help? You've come to the right place! We have a few different ways you can get help with Solarius products.

## Discord

The best way to get help is to join our Discord server.

- [Join the Discord server](https://discord.gg/000000)

## Email

We also have an email address you can use to get help.

- [Send an email](mailto:support@solarius.me)

## File a support ticket

Framework users can file a support ticket to get help with anything Solarius.

- [File a support ticket](https://framework.solarius.me/support)
